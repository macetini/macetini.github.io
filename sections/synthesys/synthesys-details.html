Overview

The Synthesys system fully encompasses work records, presentation, organization and sale of travel products and services, supported by all of the specific and often unpredictable situations that follow a complex and dynamic operation of travel agencies,and provides high automation and integration of data and business processes with the direct complicity with the surrounding IT systems in real time.

Responsibilities

  • Designing and developing front end using Adobe Flex Framework with PureMVC Pattern Framework
  • Creation of Flex custom components
  • Creation of XML parsing algorithms
  • Flexicious Data Grid integration
  • Flex based open source components integration
  • Server - Client communication logic
  • Debugging Java back end code
  • Debugging PL/SQL output
  • Communicating with project designers
  • Designing and developing small multi - platform PhoneGap application using jQuery Mobile, CSS and HTML

Mobile Application

As part of my work on the Synthesys project, I also developed a small multi-platform prototype mobile application using PhoneGap, jQuery Mobile, CSS, and HTML. It allows users to capture images of barcode and upload the result to an external server.

The application never went into production, but it was a valuable experience in mobile app development and cross-platform compatibility. The source code of is available on GitHub.
