Overview

The Shield Shader project focuses on creating a realistic shield effect using advanced shader techniques. This shader simulates the appearance of a protective energy shield, complete with dynamic lighting, transparency, and distortion effects.

Key features of the Shield Shader include:

  • Fresnel / rim lighting control for edge glow
  • Base color / tint + emissive color support
  • Distortion / refraction-like view-dependent offset
  • Procedural noise / animated scroll for energy movement
  • Impact ripple / shockwave parameters for hit feedback
  • Normal map support for surface detail
  • HDR emission friendly (works with URP post-processing)
  • Low-cost variant options for mobile/low-end targets

The shield is interactive, responding to player actions (mouse clicks) by generating impact ripples and shockwave. The shader is designed to be flexible and adaptable, allowing for customization of various parameters, such as color, transparency, and distortion.

Customization

Users can customize the shield's appearance by adjusting parameters like base color, emissive intensity, distortion strength, and ripple speed. This allows for a wide range of visual styles, from subtle energy fields to vibrant, glowing shields.

Implementation Details

The shield is actually two meshes: an outer shield mesh with the shield shader applied, and an inner mesh with a simple emissive material to give the appearance of a glow. This way the shield looks good from all angles and eliminates the artifacts that can occur with rendering inside of a transparent materials.

The URP Shader Graph and a possible Lit version are planed but not fully implemented yet. Some issues still persist, but will be resolved.

For now Unlit Custom shader is used.
