Zoomomba

October 2014 - July 2017

Zoomomba was a social multiplayer online game developed by Bigpoint GmbH using Flash and ActionScript 3. The game is currently inactive, but it was a fun project to work on. We gave it all before it went offline in 2017.

I worked on the development of the game's front-end features using ActionScript 3 and Flash, collaborating closely with the development team.
