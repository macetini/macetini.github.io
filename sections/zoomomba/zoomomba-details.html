Overview

ZooMumba was a free-to-play zoo simulator developed and published by Bigpoint where players took up the responsibility of creating and maintaining a zoo.Players started with an empty piece of land to transform into a successful zoo. Players had a variety of objects and tools at their disposal to transform the empty piece of land into one of the worldâ€™s best zoos

My role was to optimize the game for performance, implement new content and fix bugs

Responsibilities

  • Modification and improvement of client ActionScript3 code with Apache Flex framework.
  • Debugging and analyzing PHP server side code.
  • Implementation of PHP configs.
  • Modification of HTML/CSS/JS layer.
  • Modification of MySQL database data and creation of new Tables.
  • Addition of multiple smaller flash features.
  • Simple Image manipulation with Photoshop and Gimp.
  • Implementation/modification of new flash content by using Adobe Flash CC.
  • Analyzing and optimizing the AS3 code with Adobe Scout.
  • Using the Jira workflow - accepting/creating tickets.
  • Communicating with the artists on the new content for the games.
  • Creation/Reintegration of new branches with both Git and SVN.
  • Syncing release versions to live/social/stage.

Ocean World feature

The Ocean World expansion was a major content update that introduced a new aquatic-themed area to the game, allowing players to create and manage underwater exhibits and attractions.

My role in this feature was to implement new aquatic animals, design and develop underwater exhibits, and create interactive elements for players to engage with.
