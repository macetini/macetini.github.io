Overview

Seafight is an online multiplayer pirate-themed game developed by Bigpoint Games. As part of the development team, I contributed to various aspects of the game's architecture, gameplay mechanics, and user interface enhancements. My role involved collaborating with designers and other developers to create engaging content and ensure a smooth gaming experience for players.

Responsibilities

  • Modification and improvement of client ActionScript3 code with Apache Flex framework.
  • Development of PHP Website.
  • Modification of HTML/CSS/JS layer.
  • Modification of MySQL database data .
  • Addition of multiple smaller flash features.
  • Simple Image manipulation with Photoshop and Gimp.
  • Implementation/modification of new flash content by using Adobe Flash CC.
  • Analyzing and optimizing the AS3 code with Adobe Scout.
  • Using the Jira workflow - accepting/creating tickets.
  • Communicating with the artists on the new content for the games.
  • Creation/Reintegration of new branches with Git.
  • Syncing release versions to live/social/stage.

Puzzle Feature

One of the notable features I worked on was a Puzzle system within the game. This feature allowed players to engage in puzzle-solving activities, adding an element of strategy and challenge to the game experience.
