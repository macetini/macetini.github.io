Overview

Haxe Doodle Jump is a recreation of the classic mobile game "Doodle Jump," developed using the Haxe programming language and the OpenFL framework. This project demonstrates my ability to work with cross-platform development tools, implement physics-based gameplay, and create engaging user experiences on both desktop and mobile platforms.

The Main class is responsible for initializing the game and managing the main game loop.
