Overview

In the early 2010s, I created a custom Unity shader to deform 3D mesh vertices along a defined spline, enabling dynamic, interactive bending.

The spline is user-defined within the Unity Editor, allowing for real-time modifications and animations.

The spline curve is then used to deform a mesh, which is populated with a specified prefab. The shader then deforms the mesh vertices to align along the curve, allowing for real-time modifications and animations.

Both the Spline and Prefab are configured in the Unity Inspector using the UVCurve.cs component.

Users can populate and configure the spline with prefabs as needed.

Static Curve

The result is a model composed of meshes deformed to match the static curve.

This shader was an unreleased experimental feature. Although the project was modernized to Unity 2025, no further development is planned, as Unity now offers a built-in Spline System.

Dynamic Curve

Beyond a static curve, the shader can also create the illusion of movement along the path.

Crucially, the camera is static. The illusion is created by the curve deforming the model around the origin while simultaneously moving the meshes toward the camera.
