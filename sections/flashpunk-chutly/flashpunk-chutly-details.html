Overview

Flashpunk is a simple 2D game framework for Flash, built with ActionScript 3. I used it to create a small game called "Call of Chutly", a platformer where the player controls a character that must navigate through levels while avoiding obstacles and enemies.

Technologies & Links

I did all the artwork for the game, and I made an effort to only use open-source tools.

Technologies used:

  • ActionScript 3
  • Flashpunk framework
  • FlashDevelop IDE
  • Inkscape
  • Synfig Studio
  • Gimp
  • Blender

Game is still available through Ruffle emulator on Kongregate.
