Overview

A PHP + DHTML + SMARTY project originally developed in 2012 as part of a comprehensive CMS for apartment management. The project was later discontinued by the client, but has since been updated to support PHP 8.3 and MariaDB.

Features

  • Apartment listings with detailed information
  • Image galleries for each apartment
  • Reservation system for prospective tenants
  • Admin panel for managing listings and reservations

Technologies Used

  • PHP 8.3
  • MariaDB
  • SMARTY Templating Engine
  • DHTML/CSS for front-end design

I designed the Rational database myself, and constructed testing environment in VirtualBox Ubuntu server OS.
