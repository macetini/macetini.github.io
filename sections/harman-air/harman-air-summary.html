Harman Air

A collection of ActionScript 3 projects built using the Harman AIR SDK including a classic Pong game built with the MVCS architecture and a mining style game called Midas Miner both built with the Starling Framework.

As the Ruffle flash emulator for the Flash/Air platforms still do not support Stage3d live demos are not available.
